Specific APIs

An application programming interface (API) the Python library PyEnzyme, and the Java library JEnzyme, which support reading, writing, editing, merging, and visualization of EnzymeML documents.

Presently, we have three application-specific thin API layers COPASI, STRENDA DB and SABIO-RK. They are available to download in the links below:

  • TL_COPASI
  • TL_STRENDA
  • TL_BioCatNet
