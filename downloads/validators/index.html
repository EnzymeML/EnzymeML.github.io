Validators

EnzymeML has a specific validation tool that guarantees compatibility with SBML. Further application-specific validation tools have been added, such as a STRENDA DB validator to check for compatibility with the STRENDA Guidelines.

The validators are part of application-specific thin API layers COPASI, STRENDA DB andSABIO-RK. They are available to download in the links below:

  • TL_COPASI
  • TL_STRENDA
  • TL_BioCatNet
